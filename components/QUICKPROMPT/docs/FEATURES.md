### Template Variables
Each template supports dynamic variables:
- `{subject}`: Main subject of the prompt
- `{character}`: Character description or preset
- `{style}`: Artistic or photographic style
- `{lighting}`: Lighting conditions and setup
- `{mood}`: Emotional tone and atmosphere
- `{setting}`: Location and environment
